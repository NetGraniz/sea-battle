# Морской бой

Полноценная браузерная игра «Морской бой» для GitHub Pages. Проект сделан на чистых HTML5, CSS3 и Vanilla JavaScript без серверной части, фреймворков и сторонних библиотек.

## Ссылки

- Игра: https://netgraniz.github.io/sea-battle/
- Репозиторий: https://github.com/NetGraniz/sea-battle

## Правила игры

Игрок и компьютер размещают флот на поле 10x10. Цель игры - первым уничтожить все корабли соперника.

Классический набор кораблей:

- 1 корабль на 4 клетки;
- 2 корабля на 3 клетки;
- 3 корабля на 2 клетки;
- 4 корабля на 1 клетку.

Корабли нельзя ставить рядом друг с другом. Запрещено соприкосновение сторонами и углами.

Игрок ходит первым. При попадании стрелок получает дополнительный ход. При промахе ход переходит сопернику. Когда корабль уничтожен, клетки вокруг него автоматически помечаются как недоступные.

## Возможности

- два игровых поля 10x10: поле игрока и поле компьютера;
- ручная расстановка кораблей мышью;
- поворот корабля перед размещением;
- проверка корректности расстановки;
- подсветка ошибок размещения;
- случайная расстановка флота;
- очистка поля перед началом игры;
- отображение попаданий, промахов и уничтоженных кораблей;
- автоматическое определение победы или поражения;
- отображение текущего хода и важных событий;
- светлая и тёмная тема;
- автоматическое определение темы устройства;
- ручное переключение темы;
- звуковые эффекты попадания, промаха, уничтожения, победы и поражения;
- возможность отключить звук;
- адаптивный интерфейс для компьютеров и мобильных устройств.

## Уровни сложности

### Лёгкий

Компьютер стреляет полностью случайно по доступным клеткам.

### Средний

После попадания компьютер пытается добить корабль, анализируя соседние клетки.

### Сложный

Компьютер использует вероятностный поиск: анализирует размеры оставшихся кораблей и выбирает клетки с наибольшей вероятностью попадания.

## Сохранения и статистика

Все данные хранятся локально в браузере через LocalStorage.

Сохраняется статистика:

- количество сыгранных партий;
- количество побед;
- количество поражений;
- процент побед;
- текущая серия побед;
- лучшая серия побед.

Текущая партия автоматически сохраняется. После обновления страницы игру можно продолжить с того же места.

## PWA

Игра поддерживает установку как приложение на телефон или компьютер. После первого открытия доступен офлайн-режим благодаря service worker.

## Запуск локально

Проект статический. Можно открыть `index.html` в браузере или запустить любой локальный статический сервер.

```bash
python -m http.server 8000
```

После этого откройте:

```text
http://localhost:8000/
```

## Публикация на GitHub Pages

В настройках репозитория включите GitHub Pages:

```text
Settings -> Pages -> Source: Deploy from a branch -> Branch: main -> /root
```

После публикации игра будет доступна по ссылке:

```text
https://netgraniz.github.io/sea-battle/
```
